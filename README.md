# carrinho-compras

Simples carrinho de compras em React, criado com Create React App.

## Funcionalidades
- Exibir produtos
- Adicionar ao carrinho
- Atualizar/remover itens
- Exibir total

## Tecnologias
- React  
- JavaScript (ES6+)
- HTML & CSS

## Instalação
```bash
git clone https://github.com/MarianeMarinho/carrinho-compras.git
cd carrinho-compras
npm install
